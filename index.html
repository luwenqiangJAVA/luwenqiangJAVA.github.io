如何较长时间保存数据

把数据保存到客户端Cookie

Cookie客户端浏览器文件夹一个小文件，数据存储的格式key=value
使用Cookie
怎样向客户端写入Cookie
1）定义Cookie对象
Cookie cookie=new Cookie（key,value)
cookie.setAge()
response.addCookie()
怎样从客户端读取Cookie里面的内容
Cookie[] cookies=request.getCookies()
for(Cookie cookie:Cookies){
    String name=cookie.getName()
    if(value.equal(name))
        cookie.getValue()
}

Session

把数据放在服务器端，客户端把Sessionid放到cookie中JSESSIONID
1）怎样创建或获得Session
session=request.getSession(true|false)
false:如果没有返回null
true：如果没有一个新的session
request.getSession()
2）向session读写数据
session.setAttribute(String,Object)
session.getAttibute(String name)
3)怎样把session id保存到客户端中
Cookie cookie=new Cookie("JSESSIONID",session.getId())
response.addCookie(cookie)
